# Changelog

## v1.0.0

- Initial Next.js version of Lie Ledger.
- Four player cards with editable names.
- Large lie counters with + and - controls.
- Floating bottom control bar with Share, Undo, Reset and New.
- Shared game rooms by URL query string.
- API-backed persistence with local file fallback and optional Supabase storage.
- Mobile-safe fullscreen Liquid Glass UI.
