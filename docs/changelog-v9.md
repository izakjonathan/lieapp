# Scoreboard v9

- Biggest Liar banner with crown styling.
- Four quick score controls per player: -5, -1, +1, +5.
- Long-press rapid scoring on score buttons.

- Room labels in the header instead of generic shared text.
- Share/copy now uses the clean room URL.

- PWA support.

- Mobile-first iPhone layout.
